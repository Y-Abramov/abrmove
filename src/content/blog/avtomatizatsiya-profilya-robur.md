# Автоматизация продольного профиля в Топоматик Robur

Работа с продольным профилем — одна из самых рутинных операций при проектировании автомобильных дорог.
В этой статье разбираем, как ускорить этот процесс с помощью модуля ABR | Profile Tracker.

## Проблема

При работе с профилем в Топоматик Robur инженер постоянно вынужден:
- переключаться между окнами для определения текущего пикетажа
- вручную пересчитывать отметки и рабочие высоты
- запоминать или записывать значения уклонов

## Решение

ABR | Profile Tracker отображает все необходимые параметры в реальном времени при движении курсора по профилю.

Подробнее о модуле — в разделе [Мои решения](/solutions).
